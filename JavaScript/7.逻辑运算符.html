逻辑运算符一共三种: 1.与 && 只有当两个表达式都为真时,运算结果才为true. [短路操作]:当第一个表达式为false的时候,第二个表达式就不去执行,直接判断整个为假 2.或 || 只要其中一个为真,表达式就为真,两个都为假时,表达式才为假. [短路操作]:当第一个表达式为true的时候,第二个表达式就不去执行,直接判断整个为真 3.非 ! 先转换成布尔值,再取反.非运算遵从以下规律: ①操作数为空字符串 true ②操作数为非空字符串 false ③操作数是0 true ④任意非0的数值(包括infinity) false ⑤操作数是NaN true ⑥操作数是undefined true
